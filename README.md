# Harsh Jethva Portfolio

A premium personal portfolio website built with React and Vite.

## Overview

This site presents Harsh Jethva's profile as an IT Engineer, Full Stack Developer, and Cybersecurity Enthusiast. It includes:

- A bold hero section with animated visual treatment
- Skills, work, experience, certifications, and contact sections
- Responsive layouts for desktop and mobile
- Lightweight interactions built with React and Lucide icons

## Tech Stack

- React
- Vite
- Lucide React
- CSS animations and responsive design

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

- `index.html` - Vite entry HTML file
- `src/main.jsx` - Main React application
- `src/styles.css` - All styling for the portfolio

## Notes

The repository ignores generated build output and dependencies such as `dist/` and `node_modules/`.
